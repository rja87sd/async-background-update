Week 6 Day 3 Assignment | Asynchronous Code

Instructions:

Part 1: Setup Your Environment

  1. Create a new repository.
  2. Name the repository async-background-update.
  3. Initialize it with an index.html file and a script.js file.

Part 2: Implement Delay Function

  1. In script.js, define the delay function as shown in the demo: 
                    function delay(milliseconds) {
                    return new Promise((resolve) => setTimeout(resolve, milliseconds));
                }

Part 3: Async Function for DOM Manipulation

  1. Write an async function in script.js that changes the background color of the body element of the webpage.
  2. The function should change the color three times, with each color lasting for 5 seconds before switching to the next one.
  3. Utilize the delay function to control the timing.

Part 4: Testing and Validation

  1. Open your index.html in a web browser.
  2. Ensure that the background color of the body changes as expected.
  3. Verify that there is a 5-second delay between each color change.

Part 5: Deployment and Submission

  1. Commit and push your changes to the async-background-update repository.
  2. Optionally, deploy your project using GitHub Pages or another hosting service.
  3. Provide the URL link to your GitHub repository as your assignment submission.
